A sample Spring Boot application that demonstrates how to implement springdoc-openapi-starter-webmvc-ui for generating and displaying OpenAPI documentation using Swagger UI.

OpenApiConfig.java: This class is added to customize the OpenAPI documentation details.


mvn clean install
mvn spring-boot:run


Accessing Swagger UI
http://localhost:8080/swagger-ui.html



GET: http://localhost:8080/api/books

GET: http://localhost:8080/api/books/count
